# RadioBud Icon Assets

This directory contains icon assets for the RadioBud application.

## Required Files

- `icon.png` - 1024x1024px source icon (primary file)
- `icon.icns` - macOS app icon (auto-generated by electron-builder)
- `icon.ico` - Windows app icon (auto-generated by electron-builder)

## How to Add Your Icon

1. Generate your icon using AI (see prompts in project notes)
2. Save the 1024x1024px PNG as `icon.png` in this directory
3. electron-builder will automatically generate .icns and .ico files during build

## Icon Specifications

- **Size**: 1024x1024px minimum (for best quality)
- **Format**: PNG with transparency preferred
- **Color**: Tomato red (#FF6347) as primary color
- **Style**: Minimal, flat design, scales well to 16x16px

## Building with Icon

```bash
npm run build:mac        # macOS app with icon
npm run build:win        # Windows app with icon
npm run build:linux      # Linux app with icon
```

The icon will automatically be embedded in the application bundle.
